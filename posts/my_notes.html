How to do coffeescript OOP style coding

Using SRP(Single Responsibility Principle) style:

  • Create a constructor method
  • Create and call the @addEvents inside the constructor for the events of the element
  • Create methods to separate actions and for it to be readable.
  • Call the object below to run it
                  class FooBar
                    constructor: () ->
                      @$elem = $('.the-element-you-need')

                      @addEvents()

                    addEvents: () ->
                      @$elem.on 'click', (e) =>
                        e.preventDefault()
                        console.log 'Element Clicked!'

                        @magicAction()

                      $@elem.on 'hover', (e) =>
                        e.preventDefault() console.log 'Element Hovered!'

                      // Other Events

                    // Separated Action
                    magicAction: () ->
                      // Do something
                      return

                  $(document).on 'load', () =>
                    new FooBar
